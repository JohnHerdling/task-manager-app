# Task Management Application (Backend)

## Description

This project is a backend-based task management application built with Node.js and Express.

It provides a REST API that allows users to create, read, update, and delete tasks.

## Features

* Create tasks
* Retrieve all tasks
* Update tasks
* Delete tasks

## Technologies

* Node.js
* Express
* Docker
* Jest (Testing)

## Notes

This project focuses on backend functionality.
A frontend interface is not included.
